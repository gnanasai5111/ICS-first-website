LRF - Electronics

Home Catalogue LRF - Electronics

Innovation Communication Systems have vast experience in design and development of LASER Electronics products. ICS has developed multiple products for LASER RANGE FINDER (LRF). The role LRF electronics is implementation of range computation logic, controls to the LRF sub-units, interface with laser pointer and illuminator and RS-422. The detailed technical specifications LRF electronics are mentioned below.

Application

  • Measuring Target Parameters

Features

    Ranging electronics multilayer Board has following features:

  • Dimension : 90mm x 50mm x 20mm.
  • Artix 7 series FPGA (Defence grade) with compatible configuration PROM.
  • 512Kbyte Flash memory.
  • Ranging Gate circuit.
  • 30 MHZ clock with high stability.
  • Interfaces.
  • Three +5V buffered LVTTL IO APD receiver C30569-1550.
  • Laser transmitter 6 pin buffered LVTTL interface.
  • Three pin buffered LVTTL interface for laser illuminator.
  • Two pin interface laser pointer.
  • 10 pin interface for linerar motor MM-3M-F-0.5V from national aperture.
  • 6 buffered LVTTL IO’s.
  • JTAG interface, ADC,DAC.

    • Power supply board of Ranging electronics is multi layer PCB. It generate all the voltages required for the ranging electronics. It has following with the following features.

  • Dimension : 90mm x 55mm x 25mm.
  • Generates for following voltages.
    • 1.Input voltage 12V, +/-3V, 5A
    • 2.Output voltages
      • 12V, 2A should be command controlled.
      • +/-5V ,100 mA for APD Detector.
      • 30V-60V,100mA should be variable with soft start (for APD Detector).
      • 5V,50mA for TP Detector.
      • 3V,500mA laser pointer driver with command controlled.
      • 6V,200mA for H-bridge.
      • 5V,10mA.
    • 3.Rippled less than 50mV
    • 4.All voltage have isolated ground
    • 5.Power supply has proper EMI/EMI line filter at input and reverse polarity protection
