Turnkey Project Execution

Research »  Turnkey Project Execution

ICS executed a number of turnkey projects in data communication software development, embedded systems design and application software development for major public sector undertaking and private sector organizations. A brief summary of some of the major projects is given below.

  • DEVELOPMENT OF FAX SERVER SOFTWARE
    • The PC based fax server has been developed to receive and transmit fax messages in a VSAT network. The software residing at the Hub receives the fax messages from VSAT, decodes destination address automatically and relays to the corresponding destination VSAT.
    Client Name - Bharat Electronics Limited. Bangalore.
    Development Environment - Windows 95, Visual Basic

  • VIRTUAL INSTRUMENTATION SOFTWARE
    • Virtual Instrumentation software has been developed using the DAQ card of National Instruments to carry out testing of assembled PCBs in large quantities and an automatic test program will test the PCBs and generate the test report.
    Client Name - Electronics Corporation Of India Limited, Hyderabad
    Development Environment - Virtual Instrumentation Software of National Instruments (Labview) on Windows 95, Visual Basic

  • DEVELOPMENT OF PROTOCOL CONVERTER SOFTWARE
    • The Protocol Converter converts the Ethernet packets to proprietary satellite protocol. It has two interfaces Ethernet and RS232. It captures the Ethernet packets from the Ethernet LAN, converts them into proprietary satellite network packets, and sends to the Satellite Modem via RS 232 port. Similarly it takes packets from the Satellite modem and converts them to Ethernet Packets. Built-in routing mechanism identifies the source and destination by verifying the IP address of the TCP/IP packets.
    Client Name - Bharat Electronics Limited. Bangalore>
    Development Environment - Windows 9X / Windows NT, C and VC++

  • DEVELOPMENT OF ANTENNA INFORMATION MANAGEMENT SYSTEM
    • Antenna Information Management System facilitates searching of the various sub-systems of specified antenna types, specifications, drawings, Auto CAD drawings, JPEG files, PDF file, cost calculator and Pricing Information. It has a multi-user capability with password protection.
    Client Name - Antenna Products Division of Electronics Corporation Of India Limited, Hyderabad
    Development Environment - Windows 9X, Visual Basic and MS.Access

  • MESSAGE COMMUNICATION SOFTWARE
    • The media of communication used here is a Satellite link or a Multi Access Radio Telephony (MART). The satellite link is used for communicating between two headquarter police stations and MART for communicating between a headquarter police station and a remote police station.
    The MCS will reside on a stand-alone PC, which will be interfaced to VSAT through RS232 ports. The MCS will support complete X.28/X.3 to setup a SVC (Switched Virtual Circuit) with a remote DTE and to transmit message. The message communication software shall allow the initiation of X.25 call from the DTE. To forward a message from police head quarters to the remote police station MART link is used.
    Client Name - Bharat Electronics Limited, Ghaziabad
    Development Environment - Windows XP, VC++ and MS Access Database

  • AUTOMATION SOFTWARE FOR DCMS
    • Manual operations in the Accounts and stores sections pose many problems in preparing of reports, maintaining the sales, purchases and stocks positions of the society and its branches, co-operatives. The product aims to overcome the difficulties, because of manual operation.
    The product aims at the computerization of the total DCMS activities. It starts from the core accountancy and integrates all other activities including pay roll processing, purchase and sales operations and generation of reports of various kinds related to various products.
    Client Name - District Co-operative Marketing Society, Hyderabad
    Development Environment - Windows 9x, Visual Basic and MS Access Database

  • UDP TRANSMITTER / RECEIVER
    • The UDP Protocol can be used to transmit data between the central and remote locations, as it doesn't require any acknowledgement from the remote. The UDP Tx software will be on the central system for transmitting a file/directory from central to remote locations. The UDP Tx Software will transmit the selected file / directory as UDP packets. All the remote systems, which have the same multicast IP address, will receive the transmitted UDP packets. At the remote locations the UDP Rx Software will receive these UDP packets and recreates the Original file / directory.
    Client Name - Bharat Electronics Limited, Bangalore
    Development Environment - Windows 9X, JDK 1.4.

  • POLNET Broadcast Software
    • The Polnet Broadcast software facilitates sending data files/messages from central location to all police stations in broadcast mode through a VSAT network in packet switching mode.
    Client Name - Bharat Electronics Limited

  • Return Channel Software for APNET
    • This software is facilitiates the availability of a return channel can enable data acknowledgement and data request to be sent from the remote sites to the APNET hub. Remote Sites Type 2B in APNET is equipped with a PC and DVB receiver card.
    Client Name - Bharat Electronics Limited

  • SIP Stack Software Download Brochure
    • SIP Stack for SIP Client (SSSC) is one of the modules of Subscriber Terminal Secrecy Unit (STSU) to make end- to-end communication using SIP Messages as per RFC 3261.
    The SSSC implementation contains the APIs for all SIP request Messages like REGISTER, INVITE, CANCEL, BYE, ACK, SDP Payload Parser, timeouts as well as Response Message Generation for end-to-end communication. SSSC also Supports
    • Digest Authentication as per RFC 2617 for REGISTER Message.
    • Call forwarding and third party busy.
    • Processing with single as well as multiple SIP Servers.
    Client Name - Bharat Electronics Limited

    Development Environment - BREW SDK 3.1.4 or later, Microsoft Visual Studio IDE, BREW supported CDMA Mobile Handset and SIP Server

  • Subscriber Dynamic Locking System
    • The SDLS facilitates STD/ISD locking /unlocking facility for strowger / crossbar exchanges. The entire software for call processing, billing and administration of the system was developed using a MOTOROLA processor.
    Client Name - Kenya Posts & Telecommunications Corporation. Nairobi.

  • Analysis and Recording Module Download Brochure
    • The Analysis and Recording Module Model No. (ICS-ARM-01) (ARM) is a one-stop solution for Narrow Band Communication Signal Analysis. In addition to the analysis of Communication Signals at IF/AF level, the enhanced features of ARM lie in its capability to recognize, demodulate and decoding of variety of digital communication signals to assist the gathering of communication intelligence from unknown and non-cooperative transmitters/emitters. Recording/Replay of Signals at IF/AF level with time stamping and annotation proves the ARM Operator-friendly and finds wider range of usage in military and paramilitary applications. ARM is designed and engineered around a Rugged Power PC Processor meeting the stringent environmental and EMI/EMC specifications for ground based, ship borne and airborne applications.Modularity in the design of ARM enables the stacking of multiple modules to enhance the analysis of multi channel and multi signal environment.
    Client Name - DLRL – Defence Laboraties
    Development Environment - VxWorks – 6.5, WindRiver WorkBench 2.5, C, VC++, RAD-2, PBOD

  • Burst Generator / Tester Software - For defence applications

  • Coder Decoder Software - For defence applications

  • Interactive Voice Response System - Telebanking Software
    • The Application software is developed for knowing the Bank balance in the Customer account. The Customer dials a dedicated telephone Number and dials language option, his Bank account type and his account number with a password The System connects to Database and retrieves the required information and announces it to the Customer.
    Client Name - Jalagaon Janasakthi Co-Operative Bank Limited, Jalagaon

  • Interactive Voice Response System - Telebanking Software
    • It is 2 Port IVR Application Software developed to obtain the Bank balance in the Customer account. The Customer dials a dedicated telephone Number and dials language option, his Bank account type and his account number with a password. The System connects to Database and retrieves the required information and announces it to the Customer. The System also dials the predefined Telephone Numbers and reminds the Customer to pay loans in time.
    Client Name - Devangiri Bank, Auranagabad, India.

  • Help line System for Income Tax Department – Govt. of India
    • It is 4 Port IVR Application software, developed for the Help Line. The citizens can dial the IVR number know the Present Income Tax Rules regulations and tax structure for Salaried class and Business class, Tax Structure
    Client Name - Income Tax Department, Hyderabad, Vijayawada & Vishakapatnam

  • Complaint Handling System
    • IVR Application software developed for registering the faulty telephones. The system automatically registers the complaint and generates a docket number for reference. The system also records messages in the voice mail box.
    Client Name - South Central Railway, Hyderabad, India

  • Round the clock Gas Booking System
    • It is 2 port IVR Application software developed for registering gas booking over the telephone line. The system automatically registers the booking and generates a docket number for reference.
    Client Name - H.P.Gas( Vijayanagar Gas Agencies) Hyderabad, India

  • Results Announcement System
    • It provides information to the students about entrance examination results, interview dates and course fee details. When a user dials the number, the UHIS, receives the call and plays a welcome message and prompts for register number.
